# Gerald K. Portfolio

A modern, responsive personal portfolio website built with **Next.js 13**, **TypeScript**, and **Tailwind CSS**. This portfolio showcases my projects, skills, and provides an easy way for visitors to contact me or download my resume.

---

## 🚀 Features

- **Responsive Design:** Works on desktop, tablet, and mobile.
- **Dark / Light Mode Toggle:** Users can switch between dark and light themes; preference is remembered.
- **Downloadable Resume:** Visitors can download my PDF resume directly from the Hero section.
- **Projects Section:** Showcase of my projects with descriptions, tech stack, and links to GitHub or live demos.
- **Contact Form:** Fully functional contact form integrated with **EmailJS** for sending messages securely.
- **Smooth UX:** Polished UI with hover effects, focus states, and subtle animations.

---

## 🛠 Tech Stack

- **Frontend:** Next.js 13 (App Router) + TypeScript
- **Styling:** Tailwind CSS
- **Form Handling:** EmailJS (serverless API route)
- **Icons:** Lucide React
- **Hosting:** Vercel (recommended)
